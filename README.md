# ChessMp

# Running locally
Run peerjs server:

```
npm i -g peer
npm run peer
```

Serve angular with `npm start`

# Running tests

For wsl, you will need to setup `CHROME_BIN` to point to a chrome binary
```
export CHROME_BIN=/mnt/c/Program\ Files\ \(x86\)/Google/Chrome/Application/chrome.exe
```
